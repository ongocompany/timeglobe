
# 역사 지도 최상단 줌 - 시대별 핵심 국가 목록

## 선정 기준 (역사학자+지리학자 관점)

1. **지정학적 중요성** - 해당 시기 세계 정치·군사의 핵심 축
2. **문명적 파급력** - 주변 지역에 영향을 준 문화 중심
3. **지리적 균형** - 전 지구를 커버하는 분산 배치
4. **교과서 인지도** - 고등학생 수준에서 "들어봤다" 싶은 수준

## 지도 구현 시 추가 권장 사항

**줌 레벨별 분리 전략:**

- **최대 줌아웃 (전지구)** → 위 목록에서 각 시기 상위 8~12개만. 지역별 최강자 1개씩
- **중간 줌** → 위 전체 목록 + 주요 속국/종속국 (예: 로마 시대의 파르티아, 쿠샨 추가)
- **최대 줌인** → 도시국가, 지방 왕조, 부족 연맹 등

**지역 균형을 위해 반드시 포함할 "대표 문명" 슬롯:**

- 동아시아 (중국 왕조) 1개
- 인도 아대륙 1개
- 중동/메소포타미아 1개
- 지중해/유럽 1개
- 아프리카 1개 (악숨, 말리, 가나, 쿠시 중 시기별)
- 중앙아시아 1개 (흉노, 몽골, 티무르 등)
- 아메리카 1개 (1200년 이후부터)

이 구조면 최상단 줌에서도 전 지구 문명이 고르게 표현되면서 교과서 수준의 인지도를 확보할 수 있을 거야.
