## 핵심 원칙

### 원칙 1: 라벨 = 그 역사적 실체의 한국어 통칭

**판단 기준: "한국 세계사 교과서에서 뭐라고 부르는가?"**

- 지도 라벨에는 **한국인이 그 시대 그 나라를 부르는 이름**을 사용한다
- 정식국호(official name)가 아니라 **통용되는 짧은 이름**이 기본이다
- 정식국호, 이칭, 원어명, 동시대 명칭은 **툴팁/상세정보**에서 제공한다

### 원칙 2: 지도 언어 = 한국어 고정

- 한국어 사용자 대상 역사지도이므로, **한국어 통칭이 기본 라벨**
- 한국어 통칭이 없는 경우: 영어명 → 원어명 순으로 fallback
- 분쟁/민감 지역은 별도 표기 정책 명시 (향후 필요시)

## 적용 규칙

### 규칙 A: 국가 연속성이 있는 경우 → 통칭 통일

정체(왕정/공화정/제국 등)만 바뀌었을 뿐 **같은 나라로 인식되는 경우**,
전 시기에 걸쳐 하나의 한국어 통칭을 사용한다.

| 시대별 정식국호 | 라벨 |
|----------------|------|
| Kingdom of England → Kingdom of Great Britain → United Kingdom of Great Britain and Ireland → United Kingdom | **영국** |
| Kingdom of France → French Republic → French Empire → ... | **프랑스** |
| Kingdom of Greece → Hellenic Republic → ... | **그리스** |

### 규칙 B: 별개의 역사적 실체 → 고유 통칭 유지

한국어에서 **별도의 이름으로 부르는 역사적 실체**는 현대 국가명으로 통합하지 않는다.

| 역사적 실체 | 라벨 | 현대 국가명으로 퉁치면? |
|------------|------|----------------------|
| 청나라 | **청나라** | ~~중국~~ (이상함) |
| 오스만 제국 | **오스만 제국** | ~~튀르키예~~ (이상함) |
| 로마 제국 | **로마 제국** | ~~이탈리아~~ (이상함) |
| 조선 | **조선** | ~~한국~~ (이상함) |
| 무굴 제국 | **무굴 제국** | ~~인도~~ (이상함) |

### 규칙 C: 정식국호가 통칭인 경우 → 그대로 사용

일부 역사적 실체는 정식국호 자체가 한국어 통칭이다.

| 예시 | 설명 |
|------|------|
| 비잔티움 제국 | "동로마 제국"도 통용되나 "비잔티움 제국"이 교과서 표준 |
| 신성 로마 제국 | 정식국호가 곧 통칭 |
| 대한제국 | 정식국호가 곧 통칭 |

## 라벨 결정 플로우

```
1. 한국 세계사 교과서에서 이 나라를 뭐라고 부르는가?
   → 그 이름이 라벨

2. 교과서에 안 나오는 소국/단명 국가는?
   → 한국어 위키백과 표제어 참조
   → 없으면 영어명 사용

3. 같은 지역의 시대별 이름이 다른가?
   → 별개 실체면: 각각 고유 라벨 (조선, 대한제국, 대한민국)
   → 같은 나라면: 통칭 통일 (영국, 프랑스)

4. 판단이 애매하면?
   → "한국인이 이 나라 이름을 들었을 때 바로 아는가?" 테스트
```

## 데이터 필드 매핑

| 필드 | 용도 | 예시 (영국, 1850년) |
|------|------|-------------------|
| `name_ko` (라벨) | 지도에 표시 | 영국 |
| `name_en` | 툴팁/영어 참조 | United Kingdom of Great Britain and Ireland |
| `name_local` | 원어명 (향후) | - |
| `official_name` | 정식국호 (향후) | United Kingdom of Great Britain and Ireland |
