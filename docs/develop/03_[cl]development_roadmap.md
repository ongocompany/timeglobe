# TimeGlobe 단계별 개발 로드맵

> **개발 방침: Design-First (디자인 먼저, 코딩은 나중에)**
> UI/UX 디자인을 먼저 완성한 뒤 코딩에 들어가는 방식. 뒤늦은 기능 추가로 인한 코드 수정 리스크를 원천 차단한다.

---

## Phase 0: 프로젝트 인프라 세팅 (기반 공사)
> 코딩 전에 프로젝트 뼈대와 핵심 기술이 돌아가는지 확인하는 단계.

- [x] Next.js 프로젝트 초기화 (TypeScript 기반)
- [x] CesiumJS + Resium 설치 및 기본 3D 지구본 렌더링 확인
  - [x] SkyBox(우주 배경) 커스텀 적용 테스트
  - [x] 기본 마커(Entity) 1개 찍어보기
- [ ] Supabase 프로젝트 생성 및 SQL 스키마 배포
  - [ ] `20260224174800_initial_schema.sql` 적용
  - [ ] 테스트 데이터 1~2건 삽입 확인
- [ ] Git 저장소 구조 정리 (`.env` 세팅, Supabase 키 관리)
- [ ] Vercel 배포 연결 (빈 프로젝트라도 배포 파이프라인 확보)

## Phase 1: UI/UX 디자인 (Design-First)
> **코드를 한 줄도 쓰지 않고**, 모든 화면의 와이어프레임/목업을 먼저 완성하는 단계.
> 이 단계에서 "아, 이 기능도 필요하네!"를 미리 다 발견하는 것이 목표.

### 1-1. 디자인 시스템 정의
- [ ] 컬러 팔레트 확정 (다크 모드 우주 테마 기반)
- [ ] 타이포그래피 선정 (Google Fonts: 한글 + 영문)
- [ ] 마커 카테고리별 색상/아이콘 컨벤션 확정
  - 정치/전쟁(빨강), 인물/문화(노랑), 과학/발명(파랑), 건축/유물(초록), 자연재해/지질(회색)

### 1-2. 핵심 화면 와이어프레임/목업
- [ ] **메인 화면 (3D 지구본 + 타임라인)**
  - [ ] 지구본 영역, 타임라인 슬라이더 위치/크기
  - [ ] 사이드 패널 또는 하단 패널 레이아웃
  - [ ] 시대 전환 시 UI 변화 (배경색, 텍스트 등)
- [ ] **마커 클릭 → 팝업 모달**
  - [ ] 헤더 (제목, 연도, 카테고리 태그)
  - [ ] 본문 (이미지, 요약, 현재 국가명)
  - [ ] 하단 액션 (외부 링크, 도감 스탬프, 퀴즈 버튼, 관련 이벤트)
- [ ] **타임 트래블러 패스포트 (도감)**
  - [ ] 도감 전체 목록 뷰
  - [ ] 획득/미획득 마커 표시 방식
  - [ ] 진척도 바 / 통계 영역
- [ ] **미니 퀴즈 화면**
  - [ ] 질문 + 선택지 레이아웃
  - [ ] 정답/오답 피드백 UI
  - [ ] 토큰 획득 연출
- [ ] **유저 프로필 / 레벨 현황**
  - [ ] 레벨 배지 (초보 타임트래블러 ~ 타임로드)
  - [ ] 토큰 보유량 표시
  - [ ] 학습 진척도 요약 (학부모 뷰)
- [ ] **로그인 / 온보딩 화면**
- [ ] **반응형(모바일) 레이아웃**

### 1-3. 디자인 리뷰 & 확정
- [ ] 진형(jn) 최종 디자인 승인
- [ ] 디자인 기반으로 컴포넌트 목록 정리 (코딩 준비물)

## Phase 2: 프론트엔드 구현
> Phase 1에서 확정된 디자인을 기반으로 코드를 작성하는 단계.

### 2-1. 공통 기반
- [ ] 디자인 시스템 CSS 구현 (컬러, 타이포, 공통 컴포넌트)
- [ ] 라우팅 구조 세팅 (Next.js App Router)
- [ ] 다국어 처리 기본 구조 (`jsonb` 데이터 → 언어 선택)

### 2-2. 3D 지구본 & 타임라인
- [ ] CesiumJS 지구본 메인 렌더링 (SkyBox, 카메라 세팅)
- [ ] 타임라인 슬라이더 구현 (연도 이동 → 시대/마커 전환)
- [ ] 마커 렌더링 (카테고리별 색상/3D 모델)
- [ ] GeoJSON 국경선 렌더링 (CDN에서 fetch)
- [ ] Lazy Loading (시대별 마커/국경선 분할 로딩)

### 2-3. 팝업 모달 & 상호작용
- [ ] 마커 클릭 → 모달 팝업 컴포넌트
- [ ] 모달 내 데이터 바인딩 (Supabase Events 데이터)
- [ ] 관련 이벤트 추천 표시
- [ ] 외부 링크 연동

### 2-4. 사용자 기능
- [ ] Supabase Auth 로그인/가입 (소셜 로그인 포함)
- [ ] 도감(패스포트) 화면
- [ ] 미니 퀴즈 화면 + 토큰 획득 로직
- [ ] 유저 프로필 & 레벨 표시
